Goal:
Implement a bounded mind-body separation system so an actor's mind and body can persist apart under protected suspension, anchored imprisonment, and explicit restoration rules.

Scope:

* support separation states where body and mind persist as distinct runtime surfaces
* support protected body suspension with reduced environmental vulnerability while separation remains active
* support external anchors or devices that maintain, extend, or gate release from the separated state
* support actor-qualified or rule-qualified release conditions instead of universal recovery by contact alone
* support anchored body possession where an invader maintains continuity through an external vessel while cycling through disposable hosts, distinguishing anchor safety, current host exposure, and trapped-host subjective state as separate tracked surfaces
* support the trapped-host interior as an experientially active prison-like state rather than collapsing takeover to a flat dominated or stunned flag
* support reduced-power projection shells so apex antagonists can project through disposable avatars rather than risking their true body, with shell defeat remaining distinct from core neutralization
* support remote reserve resources and later reentry into stronger forms after shell defeat
* support restoration paths that return the mind and clear embedded domination or insanity while preserving the target
* support revived function returning operational capacity without necessarily closing the original fatal wound, separating functional restoration from cosmetic or anatomical repair
* support warding and target resistance as stacking separate permission checks for invasive transfer rather than one unified defense check
* connect the layer to cognition, medical restoration, and sentient-object systems where appropriate

Constraints:

* deterministic only
* no universal astral-sandbox simulation
* no assumption that all separation states self-resolve on timer expiry alone
* no assumption that shell defeat equals core neutralization for projected entities
* no assumption that functional restoration requires cosmetic or wound repair
* prefer compact separation, anchor, and restoration rules over sprawling metaphysical ontologies
* keep body state, mind state, anchor safety, host exposure, trapped-host subjective state, and release path inspectable enough for debugging and counterplay

Acceptance criteria:

* at least one actor can enter a state where body and mind are mechanically distinct
* at least one external anchor maintains or gates that state
* at least one restoration path returns the mind and clears at least one embedded mental-control condition
* at least one projection shell is defeated without neutralizing the core, which survives in remote reserves
* at least one restoration returns operational capacity while leaving visible wound damage unrepaired
* at least one anchored possession tracks anchor safety, current host exposure, and trapped-host subjective state as separate surfaces
* at least one warding check and one resistance check stack independently as permission gates for invasive transfer
* targeted tests cover deterministic separation, anchor maintenance, qualified release, projection shell defeat, restoration, and stacking permission checks
